kadai02_janken_rich2

## ②課題内容（どんな作品か）

大相撲ができるゲームです
1.「力士くじ」ボタンをクリックするとあなたの力士が決まります
2.「いざ勝負」ボタンをクリックすると相手の力士が決まります
3.相手の力士よりあなたの力士の番付が上であれば勝ち、下であれば負けです
